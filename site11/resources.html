▸ The Arsenal

Resources

Everything the sect actually uses — learning platforms, tools, suppliers, and communities.

▸ Learning

Where to Learn

Free
Arduino Official Tutorials
The best starting point for microcontrollers. Clear, well-structured, covers everything from blinking an LED to advanced projects.
 Free
MicroPython Docs
Python on embedded hardware. Great for ESP32 and Raspberry Pi Pico projects.
 Paid
IoT Specialization — Coursera
UC San Diego's IoT course. Solid fundamentals for building connected devices.
 Free
GreatScott! — YouTube
Electronics explained properly. One of the best channels for practical circuit knowledge.
 Free
Paul McWhorter — YouTube
Deep Arduino and Raspberry Pi tutorials. Very beginner-friendly, very thorough.
 Free
Hackster.io
Community project hub. Great for finding inspiration and step-by-step builds from other makers.
▸ Tools & Software

Software We Use

Software
Arduino IDE
The standard for writing and uploading code to Arduino boards. Free, cross-platform.
 Software
KiCad
Free PCB design software. Steep learning curve but incredibly powerful once you get it.
 Software
Fusion 360
CAD and CAM in one. Free for personal use. Essential for designing parts that actually fit.
 Software
PrusaSlicer
Our preferred slicer for 3D printing. Works with most printers, solid print profiles built in.
 Software
Fritzing
Draw circuit diagrams the easy way. Great for documenting your builds in a visual format.
 Software
VS Code
Best general-purpose code editor. The PlatformIO extension makes embedded development much smoother.
▸ Where to Buy Parts

Suppliers

Supplier
DigiKey
Probably the best electronics supplier for serious builds. Huge selection, reliable stock, fast shipping.
 Supplier
Adafruit
Great for breakout boards, sensors, and anything maker-focused. Their tutorials are excellent too.
 Supplier
SparkFun
Similar to Adafruit — good quality components with solid documentation.
 Supplier
AliExpress
Much cheaper on motors, sensors, and basic components. Slower shipping but hard to beat the price for prototyping.
 Supplier
Amazon
Fast shipping when you need something tomorrow. Quality varies — read reviews carefully.
